# ⌨️ My Pretty Keyboard ⌨️

This web application have to goal to help people visualize their future customized keyboard.

Currently live [here](https://my-pretty-keyboard.vercel.app/)

# 🚀 Get started 🚀

Install dependencies

```
npm install
```

Run the project

```
npm run dev
```
